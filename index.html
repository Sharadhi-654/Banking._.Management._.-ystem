Trigger Testing Sandbox TRIGGERS
Trigger 1: MinimumBalanceCheck
Fires BEFORE INSERT ON Transaction. If the transaction type is 'Withdrawal' and the resulting account balance falls below ₹1,000, it aborts the operation and throws a customized SQL error state.
Trigger 2: UpdateBalanceAfterTransaction
Fires AFTER INSERT ON Transaction. Automatically updates the parent Account's balance field based on transaction type (adds on Deposit, subtracts on Withdrawal).
